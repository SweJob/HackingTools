Labb 1:

Kan göras enskilt eller i grupp. 
    Om ni jobbar i grupp, använd gärna git(github) för att jobba tillsammans.

Installera python-nmap via PIP.

Skriv ett eget verktyg som använder nmap för att skanna ip adresser.

Krav:
    1. Möjlighet att spara resultatet av skanningen till fil (.txt)
    2. Använd input/fil för att bestämma vilka ip-adresser som ska skannas
    3. Programmet ska ha en meny där användaren kan välja vad som ska göras.

Frivilligt:
4. Använd din fantasi för att skapa fler funktioner i verktyget.

---
Skicka in skriptet, antingen via Github länk eller som en .py fil.
    Notera gärna om scriptet har testats på Windows, Linux eller Mac.
    Notera om du arbetat i grupp - Skriv alla namn på deltagare i gruppen
